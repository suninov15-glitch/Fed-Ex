# 📦 Fed-Ex Logistics Data Analysis

## 📌 Project Overview
This project analyzes a global logistics dataset including purchase orders, shipping methods, and vendor terms. The goal is to identify inefficiencies and improve delivery performance.

## ❓ Problem Statement
Delivery delays and high freight costs are impacting customer satisfaction and operational efficiency.

## 🎯 Business Objectives
- Improve delivery speed
- Reduce freight cost
- Improve vendor performance
- Increase customer satisfaction

## 🛠 Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib & Seaborn


## 📊 Key Analysis Performed
- Data cleaning (handling missing values)
- Lead time calculation
- Freight cost analysis
- Vendor performance analysis
- Shipment mode analysis

## 📈 Key Insights
- Some vendors show inconsistent performance
- High cost does not always mean fast delivery
- Certain shipment modes cause delays

## ✅ Conclusion
The analysis helps in identifying cost inefficiencies and delivery issues, enabling better decision-making in logistics operations.
